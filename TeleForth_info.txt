# Les valeurs par déaut sont initailisées à partir de la table en $fd50
#
# $1404: adresse DEFER pour ABORT
# $1406: adresse DEFER pour NUMBER

# $140e: adresse DEFER pour ?TERMINAL
# $1410: adresse DEFER pour KEY

# $1414: structure du vocabulaire SOUNDS
# $1418: structure du vocabulaire GRAFX

# $1420: structure du vocabulaire IOS
# $1424: adresse DEFER pour MINITEL

# $1430: adresse variable NEWm
# $1432: adresse variable OLDm
# $1434: adresse variable WRKm
# $1436: adresse variable org^
# $1438: adresse variable mid^
# $143a: adresse variable end^
# $143c: début zone libre
#
#
#
# Vocabulaire:
# CFA:
#	(DODOES)
# PFA:
#	(DOVOC)
#	ptr vers stucture du vocabulaire (PFA+4 pour les vocabulaires en RAM)
#	ptr vers le VOC-LINK du vocabulaire précédent (ou 00,00)
#	structure d'un vocabulaire:
#		00-01: $81;$a0 (mot nul)
#		02-03: NFA du dernier mot de ce vocabulaire
#
#
# Vocabulaire fig-forth normal:
# CFA:
#	(DODOES)
# PFA:
#	(DOVOC)
#	$81,$a0 (mot nul)
#	NFA du dernier mot de ce dictionnaire
#	ptr vers le vocabulaire précédent (ou 00,00 )
#
#

#
# Autres différences avec le Fig-Forth:
#
# Les valeurs après une instruction de type BRANCH sont des adresses absolues et
# non des valeurs relatives comme pour le Fig-Forth
